# Plan de repetición doblemente espaciada por descriptor

Cada descriptor se distribuye en 3 días. Dentro de cada día se usan tres momentos de práctica:

| Momento | Duración | Propósito |
|---|---:|---|
| Activación | 5–8 min | Recuperar lo aprendido antes |
| Entrenamiento principal | 20–30 min | Resolver preguntas nuevas y adaptativas |
| Repaso de errores | 6–12 min | Repetir fallas con preguntas gemelas |

## Día 1

- Concepto de límite
- Valor puntual vs límite
- Tablas
- Gráficas básicas

## Día 2

- Límite lateral izquierdo
- Límite lateral derecho
- Límite bilateral
- Huecos y saltos

## Día 3

- Expresiones algebraicas
- Aplicación contextual
- Síntesis
- Informe y certificación

## Regla de repetición

Una microhabilidad vuelve si:

```text
Dominio < 0.85
Recuperabilidad < 0.75
Errores recientes >= 2
Pregunta perdida o quemada
```

## Fórmula de prioridad

```text
Prioridad = 0.45(1 - dominio) + 0.35(1 - recuperabilidad) + 0.15(errores recientes) + 0.05(importancia)
```
